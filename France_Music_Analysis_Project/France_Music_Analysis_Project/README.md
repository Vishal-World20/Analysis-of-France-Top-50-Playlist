# France Top 50 Playlist Analysis

## Project Title
Audience Sensitivity, Content Compliance & Format Preference Analysis of France Top 50 Playlist

## Objective
Analyze France Top 50 playlist data to understand:
- Explicit content sensitivity
- Release format preference
- Song duration behavior
- Album structure impact

## Technologies Used
- Python
- Pandas
- Seaborn
- Plotly
- Streamlit

## Run Streamlit App

```bash
streamlit run app/streamlit_app.py
```

## Project Structure

- data/
- notebooks/
- app/
- reports/
- outputs/

## Key KPIs
- Explicit Content Share
- Clean Content Dominance Ratio
- Single vs Album Ratio
- Average Song Duration
- Album Size Impact Index
